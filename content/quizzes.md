# Quizzes and Assessments

## Quiz Formats

### Multiple Choice Questions
- **Purpose**: Test theoretical knowledge and concepts
- **Format**: 4-5 options per question
- **Time**: 1-2 minutes per question
- **Topics**: Syntax, best practices, framework features

### Short Answer Questions
- **Purpose**: Test understanding of concepts and problem-solving
- **Format**: 2-3 sentence responses
- **Time**: 3-5 minutes per question
- **Topics**: Explain concepts, compare approaches, identify issues

### Coding Exercises
- **Purpose**: Test practical coding skills
- **Format**: Write code to solve specific problems
- **Time**: 10-15 minutes per exercise
- **Topics**: Implement features, fix bugs, optimize code

### Code Analysis
- **Purpose**: Test ability to read and understand code
- **Format**: Analyze provided code and answer questions
- **Time**: 5-10 minutes per question
- **Topics**: Identify bugs, predict output, suggest improvements

## Quiz Preparation Tips

### Before the Quiz
1. **Review Notes**: Go through your class notes and readings
2. **Practice Coding**: Work through coding exercises and examples
3. **Understand Concepts**: Don't just memorize, understand the "why"
4. **Get Rest**: Ensure you're well-rested before the quiz

### During the Quiz
1. **Read Carefully**: Read each question thoroughly before answering
2. **Manage Time**: Allocate time based on question difficulty
3. **Show Work**: For coding questions, show your thought process
4. **Review Answers**: If time permits, review your answers

### After the Quiz
1. **Review Feedback**: Go through instructor feedback carefully
2. **Identify Weak Areas**: Focus on topics you struggled with
3. **Ask Questions**: Don't hesitate to ask for clarification
4. **Practice More**: Work on additional exercises for weak areas

## Sample Questions

### HTML/CSS Questions

**Multiple Choice:**
Which HTML5 element is most appropriate for marking up a navigation menu?
- A) `<nav>`
- B) `<menu>`
- C) `<navigation>`
- D) `<header>`

**Short Answer:**
Explain the difference between CSS Grid and Flexbox. When would you use each?

**Coding Exercise:**
Create a responsive navigation bar that collapses on mobile devices using CSS.

### JavaScript Questions

**Multiple Choice:**
What will be the output of the following code?
```javascript
const arr = [1, 2, 3];
const doubled = arr.map(x => x * 2);
console.log(doubled);
```
- A) [1, 2, 3]
- B) [2, 4, 6]
- C) [1, 4, 9]
- D) Error

**Code Analysis:**
Identify the issue in this code and suggest a fix:
```javascript
function fetchUserData(userId) {
  fetch(`/api/users/${userId}`)
    .then(response => response.json())
    .then(data => {
      console.log(data);
    });
}
```

### Vue.js Questions

**Multiple Choice:**
Which Vue.js directive is used for conditional rendering?
- A) `v-if`
- B) `v-show`
- C) `v-for`
- D) `v-bind`

**Short Answer:**
Explain the difference between `v-if` and `v-show`. When would you use each?

**Coding Exercise:**
Create a Vue component that displays a list of items with the ability to add new items.
